Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
В функции test(), defer func() { x++ }() увеличивает значение x на 1 после того, как функция test() завершила свое выполнение, но перед тем, как она вернула свой результат. Поскольку x возвращается как именованный результат, его значение увеличивается на 1 перед возвратом, поэтому test() возвращает 2.

В функции anotherTest(), defer func() { x++ }() также увеличивает значение x на 1, но после того, как x был возвращен. Поскольку x возвращается как неименованный результат, его значение не изменяется после возврата, поэтому anotherTest() возвращает 1.

2
1
```
